### Image Compressor

In this assignment, let's build a Image Compressor  by applying the concepts we have learned so far.

#### Reference

[https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/imagecompress.gif](https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/imagecompress.gif)

#### Functionalities to be Achieved:

- Create a graphical user interface (GUI) for the image compressor application.
- Allow users to select an input image file and an output folder.
- Provide an option to adjust the compression quality (1-95).
- Compress the input image using the selected quality and save it to the output folder.
- Display a success message with the output file path after compression.
- Handle errors and exceptions during the compression process.
- Validate user input to ensure that the input file is an image and the output folder exists.


#### Recommended Libraries/Packages

- **tkinter:** For building the graphical user interface (GUI) of the application.
- **PIL (Python Imaging Library):** For opening, compressing, and saving images.
- **os:** For creating the output folder if it does not exist.
- **filedialog (from tkinter):** For allowing users to select input files and output folders.
- **messagebox (from tkinter):** For displaying success and error messages.

These libraries provide the necessary tools to implement the functionalities. Feel free to explore and integrate other relevant libraries to enhance or customize the application as needed.
