### Todo List

In this assignment, let's build a Todo List Application  by applying the concepts we have learned so far.

#### Reference

[https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/todolist.gif](https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/todolist.gif)

#### Functionalities to be Achieved:

- Create a graphical user interface (GUI) for the Todo List application.
- Allow users to add tasks to the list.
- Display the list of tasks in a scrollable area.
- Provide an option to delete selected tasks from the list.
- Handle user input and update the task list accordingly.
- Display warning or error messages for invalid user input or other exceptions.


#### Recommended Libraries/Packages

- **tkinter:** For building the graphical user interface (GUI) of the application.
- **PIL (Python Imaging Library):** For loading and resizing images used in the application.
- **requests:** For making HTTP requests to load images from a URL.
- **BytesIO:** For handling binary data from HTTP responses.

These libraries provide the necessary tools to implement the functionalities. Feel free to explore and integrate other relevant libraries to enhance or customize the application as needed.
