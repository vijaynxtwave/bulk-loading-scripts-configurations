### Chess Game

In this project, we will create an interactive Chess Game  with a graphical user interface (GUI) using Python and Tkinter.

#### Reference

[https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/chessgame.gif](https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/programming-foundations/updated%20project%20gifs/updated%20project%20gifs/chessgame.gif)

#### Functionalities to be Achieved:

- Display a chessboard with pieces and allow players to interact.
- Enable players to make moves by selecting and clicking on squares.
- Alternate turns between white and black players.
- Provide options to start a new game, undo moves, or surrender.
- Show a message when the game ends, announcing the winner.

#### Recommended Libraries/Packages

- **tkinter:** For building the graphical user interface (GUI).
- **chess:** For handling the chess game logic and move validation.

These libraries provide everything needed to implement the game logic and GUI.
